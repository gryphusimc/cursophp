Curso de PHP

Imágenes II

Por Jesús Ruiz-Ayúcar / @chuso

Qué vamos a ver

Ya sabemos lo básico para trabajar con imágenes: recortarlas, combinarlas, escalarlas, cambiar de formato, aplicar filtros, etc.

Vamos a ver cómo podemos podemos crear con colores, trabajar con píxeles individuales, añadir textos, formas geométricas, etc.

imagecolorallocate()

Crea un color para una imagen.

int imagecolorallocate ( resource $image , int $red , int $green , int $blue )

Asignamos un valor a cada color RGB (Red Green Blue) entre 0 y 255 (o entre 0x00 y 0xFF):


<?php
// Por defecto tiene negro de color de fondo:
$imagen = imagecreatetruecolor(100, 100);

// Creamos un color blanco:
$color = imagecolorallocate($imagen, 0xFF, 0xFF, 0xFF);

// Mostramos y destruimos la imagen:
header("Content-Type: image/png");
imagepng($imagen);
imagedestroy($imagen);

Consultar un color

Si queremos consultar el color que hay en un determinado píxel podemos usar imagecolorat() y imagecolorsforindex(). La primera nos devolverá el color en forma de entero (int) y la segunda extrae el color individual de cada color RGB.

int imagecolorat ( resource $image , int $x , int $y )
array imagecolorsforindex ( resource $image , int $index )

imagesetpixel()

Asignamos el valor (color) de un pixel individual.

bool imagesetpixel ( resource $image , int $x , int $y , int $color )

imagettftext()

Escribe un texto en una imagen usando fuentes TrueType (TTF).

array imagettftext ( resource $image , float $size , float $angle , int $x , int $y , int $color , string $fontfile , string $text )

Continuando con el ejemplo anterior:


<?php
// Por defecto tiene negro de color de fondo:
$imagen = imagecreatetruecolor(100, 30);

// Creamos un color blanco:
$color = imagecolorallocate($imagen, 0xFF, 0xFF, 0xFF);

// Añadimos un texto:
imagettftext($imagen, 12, 0, 10, 20, $color, "arial.ttf", "Probando...");

// Mostramos y destruimos la imagen:
header("Content-Type: image/png");
imagepng($imagen);
imagedestroy($imagen);

El orden importa

Al dibujar sobre un lienzo (recurso gd), lo último que añadamos se situará sobre todo lo demás que hubiera. Prueba a ejecutar el código del ejemplo anterior añadiendo otro texto en la misma posición:


<?php
// Por defecto tiene negro de color de fondo:
$imagen = imagecreatetruecolor(200, 30);

// Creamos un color blanco:
$color1 = imagecolorallocate($imagen, 0xFF, 0xFF, 0xFF);
// Creamos un color amarillo
$color2 = imagecolorallocate($imagen, 0xFF, 0x00, 0x00);

// Añadimos un texto:
imagettftext($imagen, 12, 0, 10, 20, $color1, "arial.ttf", "Probando...");
imagettftext($imagen, 20, 0, 10, 20, $color2, "arial.ttf", "Otra prueba");

// Mostramos y destruimos la imagen:
header("Content-Type: image/png");
imagepng($imagen);
imagedestroy($imagen);

imageline()

Dibuja una línea recta.

bool imageline ( resource $image , int $x1 , int $y1 , int $x2 , int $y2 , int $color )

Por ejemplo:


<?php
$im = imagecreatetruecolor(300, 100);
$color = imagecolorallocate($im, 0xFF, 0xFF, 0xFF);

imageline($im, 4, 4, 295, 95, $color);
header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

imagefilledrectangle()

Crea un rectángulo con relleno.

bool imagefilledrectangle ( resource $image , int $x1 , int $y1 , int $x2 , int $y2 , int $color )

Dibujamos un rectángulo blanco:


<?php
$im = imagecreatetruecolor(300, 100);
$color = imagecolorallocate($im, 0xFF, 0xFF, 0xFF);

// Fíjate en las coordenadas:
imagefilledrectangle($im, 4, 4, 295, 95, $color);
header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

imagefilledellipse()

Crea una elipse con relleno.

bool imagefilledellipse ( resource $image , int $cx , int $cy , int $width , int $height , int $color )

Dibujamos un círculo negro:


<?php
$im = imagecreatetruecolor(300, 100);

// Pintamos un fondo blanco
$bg = imagecolorallocate($im, 0xFF, 0xFF, 0xFF);
imagefilledrectangle($im, 0, 0, 300, 100, $bg);

// Pintamos el círculo
$fg = imagecolorallocate($im, 0, 0, 0);
imagefilledellipse($im, 150, 50, 40, 40, $fg);

header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

imagefilledpolygon()

Dibuja un polígono con relleno.

bool imagefilledpolygon ( resource $image , array $points , int $num_points , int $color ) 

<?php
$im = imagecreatetruecolor(300, 300);

$bg = imagecolorallocate($im, 230, 95, 0); // Pintamos el fondo
imagefilledrectangle($im, 0, 0, 300, 300, $bg);

// Pintamos una triángulo blanco
$coordenadas = [
	150, 50,
	50, 190,
	250, 190];
$fg = imagecolorallocate($im, 0x00, 0x00, 0xCC);
imagefilledpolygon($im, $coordenadas, 3, $fg);

header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

imagefilledarc

Dibuja una curva (arco) con relleno.

bool imagefilledarc ( resource $image , int $cx , int $cy , int $width , int $height , int $start , int $end , int $color , int $style )
  • $image — recurso imagen gd
  • $cx y $cy — coordenadas del centro
  • $width y $height — ancho y alto
  • $start y $end — grado de inicio y de fin
  • $color — color de relleno
  • $style — puede ser:
    • IMG_ARC_PIE: Relleno tipo gráfico de pastel.
    • IMG_ARC_CHORD: Une inicio y fin con una recta.
    • IMG_ARC_NOFILL: Solo el borde (sin relleno).
    • IMG_ARC_EDGED: Traza líneas de inicio y final al centro.

Ejemplo de imagefilledarc()


<?php
// Por defecto tiene negro de color de fondo:
$imagen = imagecreatetruecolor(500, 350);

// Creamos un color amarillo
$color = imagecolorallocate($imagen, 0xFF, 0xFF, 0x00);

// Arcos:
imagettftext($imagen, 12, 0, 50, 45, $color, "arial.ttf", "IMG_ARC_PIE");
imagefilledarc($imagen, 100, 100, 180, 90, 50, 340, $color, IMG_ARC_PIE);

imagettftext($imagen, 12, 0, 350, 45, $color, "arial.ttf", "IMG_ARC_CHORD");
imagefilledarc($imagen, 400, 100, 180, 90, 50, 340, $color, IMG_ARC_CHORD);

imagettftext($imagen, 12, 0, 50, 190, $color, "arial.ttf", "IMG_ARC_NOFILL");
imagefilledarc($imagen, 100, 250, 180, 90, 50, 340, $color, IMG_ARC_NOFILL);

imagettftext($imagen, 12, 0, 250, 190, $color, "arial.ttf", "IMG_ARC_PIE|IMG_ARC_EDGED");
imagefilledarc($imagen, 400, 250, 180, 90, 50, 340, $color, IMG_ARC_NOFILL|IMG_ARC_EDGED);

// Mostramos y destruimos la imagen:
header("Content-Type: image/png");
imagepng($imagen);
imagedestroy($imagen);

Figuras sin relleno

Hemos visto funciones para crear figuras con un color de relleno. Tenemos las mismas funciones para crear esas figuras sin relleno y con contorno:

  • imagearc()
  • imageellipse()
  • imagepolygon()
  • imagerectangle()

Por ejemplo, un rectángulo sin relleno:


<?php
$im = imagecreatetruecolor(300, 100);
$color = imagecolorallocate($im, 0xFF, 0xFF, 0xFF);
imagerectangle($im, 4, 4, 295, 95, $color);
header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

imagesetthickness()

Por defecto los bordes son de 1px. Esta función establece un nuevo ancho ancho, que se asignará en futuras llamadas a funciones que dibujen bordes o líneas —imageline().

bool imagesetthickness ( resource $image , int $thickness )

Dibujamos el rectángulo del ejemplo anterior, pero con un borde de 3px.:


<?php
$im = imagecreatetruecolor(300, 100);
$color = imagecolorallocate($im, 0xFF, 0xFF, 0xFF);
imagesetthickness($im, 3);
imagerectangle($im, 4, 4, 295, 95, $color);
header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

Resumen

En este unidad hemos aprendido cómo trabajar a bajo nivel con imágenes:

  • imagecolorallocate(): creando colores
  • imagecollorat() y imagesetpixel(): leyendo pixeles y estableciendo colores a uno solo pixel.
  • imagettftext(): escribiendo texto TTF.
  • Formas, con y sin relleno:
    • imageline(): líneas
    • imagerectangle(): rectángulos
    • imageellipse(): elipses
    • imagepolygon(): polígonos
    • imagearc(): arcos
  • imagesetthickness(): modificar el grosor del borde
