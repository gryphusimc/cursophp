Curso de PHP

Introducción a las imágenes

Por Jesús Ruiz-Ayúcar / @chuso

libgd

Es una biblioteca externa que incluye PHP de manera predeterminada.

Esta biblioteca permite crear y manipular imágenes.

Es el estándar de facto para tratamiento de imágenes en PHP. También se utiliza mucho imagemagick.

imagecreatetruecolor()

Crea un recurso asociado a una nueva imagen (vacía) del tamaño indicado. Este recurso es de tipo imagen.

resource imagecreatetruecolor ( int $width , int $height )

Ejemplo:


<?php
// Creamos un recurso de imagen gd:
$imagen = imagecreatetruecolor(800, 600);
echo get_resource_type($imagen); // Muestra 'gd'

// También podemos crearlo así:
$imagen2 = @imagecreatetruecolor(800, 600) or die ("Error al crear imagen");

imagedestroy()

Igual que debemos cerrar los ficheros con fclose(), también debemos cerrar los recursos de tipo imagen gd después de usarlo. Para ello utilizaremos imagedestroy().

bool imagedestroy ( resource $image )

Flujo de trabajo general:


<?php
// Abrimos el recurso (creamos la imagen):
$imagen = @imagecreatetruecolor(800, 600) or die ("Error al crear imagen");

// Trabajamos con ella (la mostramos o guardamos)

// Cerramos recurso (destruimos imagen):
imagedestroy($imagen);

Otras formas de crear recursos gd

Hemos visto que podemos crear una imagen vacía mediante imagecreatetruecolor(). Tenemos muchas más formas:

  • imagecreatefromgif() - A partir de un fichero gif.
  • imagecreatefromjpeg() - A partir de un fichero jpg.
  • imagecreatefrompng() - A partir de un fichero png.
  • imagecreatefromwebp() - A partir de un fichero webp.
  • imagecreatefromstring() - A partir de un string.

Para crear un recurso gd a partir de un fichero gif:


<?php
// Abrimos el fichero
$nombreimagen = "logo.gif";
$im = @imagecreatefromgif($nombreimagen);

// Trabajamos con la imagen
// Finalmente la borramos
imagedestroy($im);

Mostrar/Guardar una imagen

De poco nos sirve abrir/crear imágenes sin hacer nada más. Vamos a ver cómo podemos guardar una imagen o cómo mostrarla al navegador. Vamos a especificar más el flujo general de trabajo:

  1. Abrimos o creamos la imagen.
  2. Trabajamos con ella.
  3. Guardamos o mostramos la imagen.
  4. Destruimos el recurso.


Tenemos varias funciones, dependiendo del formato que queramos usar.

Funciones de salida de imágenes

Podemos usar todas estas funciones para guardar o mostrar una imagen:

  • imagegif()
  • imagepng()
  • imagejpeg()
  • imagewebp()


Ejemplo que muestra cómo convertir de gif a png una imagen:


<?php
$im = @imagecreatefromgif("logo.gif");
imagepng($im, "logo.png");
imagedestroy($im);

Mostrando una imagen

PHP no solo sirve contenido tipo texto (HTML o texto plano), sino que también puede mostrar una imagen. Para ello, usaremos las mismas funciones que para guardar imágenes, pero sin indicar el nombre del fichero donde la guardaremos. Pero para que funcione debemos asegurarnos de que no mostramos nada más antes, y debemos usar la función header(), que indica al navegador que estamos sirviendo una imagen.


<?php
// echo "Hola"; // Esto haría que no funcionara
$im = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");

// Tampoco funcionaría si no llamáramos a header()
header("Content-Type: image/gif"); 
imagegif($im);
imagedestroy($im);

imagesx() e imagesy()

Obtienen el ancho y el alto respectivamente de un recurso de tipo gd.

int imagesx ( resource $image )
int imagesy ( resource $image )

Ejemplo de uso:


<?php
$im = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");

$ancho = imagesx($im);
$alto  = imagesy($im);

echo "La imagen tiene ${ancho}px de ancho y ${alto}px de alto\n";
imagedestroy($im);

imagecopy()

Copia y/o recorta una imagen, sin cambiar su tamaño.

bool imagecopy ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $src_w , int $src_h )

Parámetros:

  • $dst_im - Imagen de destino.
  • $src_im - Imagen de origen.
  • $dst_x - Coordenada x de destino.
  • $dst_y - Coordenada y de destino.
  • $src_x - Coordenada x de origen.
  • $src_y - Coordenada y de origen.
  • $src_w - Ancho.
  • $src_h - Alto.

Ejemplo de imagecopy()

Vamos a crear una nueva imagen que sea un recorte del recuadro superior izquierdo de 100x100px. Fíjate en que las coordenadas (0,0) están en la esquina superior izquierda.


<?php
$orig = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");
$dest = imagecreatetruecolor(100, 100);

imagecopy($dest, $orig, 0, 0, 0, 0, 100, 100);
header("Content-Type: image/png");
imagepng($dest);

imagedestroy($orig);
imagedestroy($dest);

imagecopyresampled()

Copia, escala y recorta una imagen. O combinar varias.

bool imagecopyresampled ( resource $dst_image , resource $src_image , int $dst_x , int $dst_y , int $src_x , int $src_y , int $dst_w , int $dst_h , int $src_w , int $src_h )
  • $dst_image - Imagen de destino.
  • $src_image - Imagen de origen.
  • $dst_x - Coordenada x de destino.
  • $dst_y - Coordenada y de destino.
  • $src_x - Coordenada x de origen.
  • $src_y - Coordenada y de origen.
  • $dst_w - Ancho de destino.
  • $dst_h - Alto de destino.
  • $src_w - Ancho de origen.
  • $src_h - Alto de origen.

Ejemplo de imagecopyresampled()


<?php
$orig = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");
$ancho = imagesx($orig);
$alto  = imagesy($orig);
$dest1 = imagecreatetruecolor($ancho, $alto);
$dest2 = imagecreatetruecolor(100, 100);
$dest3 = imagecreatetruecolor($ancho/2, $alto/2);

imagecopyresampled($dest1, $orig, 0, 0, 0, 0, $ancho, $alto, $ancho, $alto);
imagepng($dest1, "logo.1.png"); // Copiamos

imagecopyresampled($dest2, $orig, 0, 0, 0, 0, 100, 100, 100, 100);
imagepng($dest2, "logo.2.png"); // Recortamos como con imagecopy()

imagecopyresampled($dest3, $orig, 0, 0, 0, 0, $ancho/2, $alto/2, $ancho, $alto);
imagepng($dest3, "logo.3.png"); // Escalamos

imagedestroy($orig);
imagedestroy($dest1);
imagedestroy($dest2);
imagedestroy($dest3);

imagerotate()

Rota una imagen con el ángulo indicado y devuelve una nueva.

resource imagerotate ( resource $image , float $angle , int $bgd_color [, int $ignore_transparent = 0 ] )

Por ejemplo:


<?php
$im = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");
$nueva = imagerotate($im, 180, 0); // El 0 es el color negro.
header("Content-Type: image/png");
imagepng($nueva);
imagedestroy($im);
imagedestroy($nueva);

imageflip()

En PHP 5.5 o superior. Voltea una imagen en el modo indicado (vertical, horizontal o ambos).

bool imageflip ( resource $image , int $mode )

Modos:

  • IMG_FLIP_HORIZONTAL voltea horizontalmente.
  • IMG_FLIP_VERTICAL voltea verticalmente.
  • IMG_FLIP_BOTH voltea en ambos sentidos.

<?php
$im = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");
imageflip($im, IMG_FLIP_VERTICAL); // Volteo vertical
header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

imagefilter()

Aplica un filtro a una imagen.

bool imagefilter ( resource $image , int $filtertype [, int $arg1 [, int $arg2 [, int $arg3 [, int $arg4 ]]]] )

Existen unos 12 filtros, alguno de ellos permiten parámetros para configurarlos. Algunos filtros son:

  • IMG_FILTER_NEGATE - invierte los colores
  • IMG_FILTER_GRAYSCALE - convierte a escala de grises
  • IMG_FILTER_GAUSSIAN_BLUR - desenfoque gaussiano
  • IMG_FILTER_PIXELATE - pixelar (con parámetros)

<?php
$im = @imagecreatefrompng("logo.png") or die("Error al cargar imagen");
imagefilter($im, IMG_FILTER_NEGATE);
imagefilter($im, IMG_FILTER_PIXELATE, 20);
header("Content-Type: image/png");
imagepng($im);
imagedestroy($im);

Resumen

  • libgd, biblioteca gráfica.
  • imagecreatetruecolor(), crea una imagen en blanco.
  • imagedestroy(), cerrar un recurso imagen gd.
  • Crear una imagen a partir de un fichero.
  • Mostrar y guardar imágenes.
  • imagesx() e imagesy(), ancho y alto.
  • imagecopy() copiar o recortar una imagen.
  • imagecopyresampled() copar, recotar o escalar.
  • imagerotate() gira una imagen.
  • imageflip() voltea una imagen.
  • imagefilter() aplicar filtros.
