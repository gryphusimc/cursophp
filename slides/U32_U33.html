Introducción de funciones

Por Jesús Ruiz-Ayúcar / @chuso

¿Qué es una función?

  • Una "herramienta" que hace algo por nosotros.
  • A veces reciben parámetros (valores).
  • A veces retornan valores.


Por ejemplo:


$resultado = nombre_funcion($parametro1, $parametro2);


Ya hemos usado muchas funciones, como por ejemplo:


					$today = date("d/m/Y");

Pero hasta ahora para nosotros son cajas negras, dentro de las cuales sucede «magia».

Pero podemos crear nuevas funciones

No estamos limitados a las funciones del sistema. Podemos programarnos las nuestras o incluir otras desarrolladas por terceros.

¿Por qué crear una función?

  • Ahorrar tiempo — gracias a que nos permiten reutilizar código.
  • Facilitar el trabajo — un código bien dividido nos ayuda a enfocar mejor un problema.
  • Mejorar el mantenimiento — bien usadas, las funciones pueden simplificar mucho el mantenimiento de un software.

Comprobando si la función existe

No podemos definir dos funciones con el mismo nombre. Es recomendable comprobar previamente si existe.

bool function_exists ( string $function_name )

Si vamos a utilizar una función que es posible que no esté presente en el sistema, es recomendable comprobarlo antes.

Por ejemplo, al programar para WordPress, si se va a utilizar una función de un plugin, por convenio siempre se comprueba previamente si la función existe.

Clasificación de funciones

Parámetros

Podemos dividir las funciones en dos grupos:

  1. Funciones que reciben parámetros.
  2. Funciones que no reciben parámetros.


Nota: Usaremos indistintamente «parámetros» y «argumentos».


Valor devuelto

  1. Funciones que devuelven un valor.
  2. Funciones que no devuelven ningún valor.

Creación de funciones

Sin parámetros y sin retorno


function nombre_funcion() {
	/* Código de la función */
}

Por ejemplo, una función que muestre un mensaje pantalla:


<?php

// En los nombres de funciones solemos seprar palabras con _
function mostrar_mensaje() {
	echo "Mensaje\n";
}

// invocamos a la función así:
mostrar_mensaje();

Sin parámetros y con retorno


function nombre_funcion() {
	/* Código de la función */
	return expresion;
}

Por ejemplo, una función que devuelva la fecha actual en formato «dd/mm/YYYY»:


<?php
function obtener_fecha() {
	return date("d/m/Y");
}

echo "Fecha actual: " . obtener_fecha();

Con un parámetro


function nombre_funcion($nombre_param) {
	/* Código de la función */
}

Por ejemplo, una función que salude al usuario que pasamos como parámetro:


<?php
function saluda($nombre) {
	echo "Hola, " . $nombre . ".";
}

saluda("Jaime"); // Muestra "Hola, Jaime."

Con varios parámetros

Los separamos por comas:


function nombre_funcion($param1, $param2, $param3 ... ) {
	/* Código de la función */
}

Por ejemplo, para la media entre 2 números:


<?php
function media($num1, $num2) {
	return ($num1 + $num2) / 2;
}

$resultado = media(303, 404); // Devuelve 353.5;

Con parámetros opcionales

A los parámetros opcionales les asignamos un valor predeterminado:


function nombre_funcion($param1, $param_opc2 = val2, $param_opc3 = val3 ... ) {
	/* Código de la función */
}

Por ejemplo, una función que devuelva la suma de 2 o de 3 números:


<?php
function suma($num1, $num2, $num3 = 0) {
	return $num1 + $num2 + $num3;
}

$resultado1 = suma(303, 404);      // Devuelve 707
$resultado2 = suma(303, 404, 101); // Devuelve 808

Argumentos variables

Si quisiéramos aceptar un número indefinido de argumentos:

Desde PHP 5.6


function nombre_funcion(...$mas_params) {
	// $mas_params será un array con todos los demás parámetros introducidos
}

Ejemplo


<?php
function suma_variable(...$nums) {
	$total = 0;
	foreach($nums as $num) {
		$total += $num;
	}

	return $total;
}

$resultado1 = suma_variable(303);              // Devuelve 303
$resultado2 = suma_variable(303, 1, -4, 1000); // Devuelve 1300

Hasta PHP 5.5

func_get_args() devuelve un array con todos los parámetros:
array func_get_args ( void )

func_num_args() devuelve el número total de argumentos:
int func_num_args ( void )
func_get_arg() devuelve el argumento de la posición solicitada, contando desde cero:
mixed func_get_arg ( int $arg_num )

Ejemplo:


<?php
function suma_variable($num1) {
	$total = 0;
	foreach(func_get_args() as $num) {
		$total += $num;
	}

	return $total;
}

$resultado1 = suma_variable(303);              // Devuelve 303
$resultado2 = suma_variable(303, 1, -4, 1000); // Devuelve 1303

Parámetros por referencia

Al pasar una variable como parámetro, se copia su valor:


<?php
function repetir($palabra) {
	$palabra .= $palabra;
	echo $palabra;
}

$saludo = "Hola";
echo repetir($saludo); // Muestra "HolaHola";
echo $saludo; // ¿Qué muestra?

Muestra "Hola";

¿Cómo podemos hacer que se actualize el valor de la variable? Usando parámetros por referencia.

Continuación

Dos formas de pasar parámetros a una función:
  1. Por valor — Es la forma predeterminada. Se copia el valor.
  2. Por referencia - Se copia una referencia de la variable; si la modificamos, el cambio será visible fuera de la función.

<?php
function repetir(&$palabra) { // Fíjate en el símbolo &
	$palabra .= $palabra;
	echo $palabra;
}

$saludo = "Hola";
echo repetir($saludo); // Muestra "HolaHola"
echo $saludo; // Ahora sí, muestra "HolaHola"
