Funciones del sistema

Por Jesús Ruiz-Ayúcar / @chuso

¿Qué es una función?

  • Una "herramienta" que hace algo por nosotros.
  • A veces reciben parámetros (valores).
  • A veces retornan valores.


Por ejemplo:


$resultado = nombre_funcion($parametro1, $parametro2);


Ya hemos usado una función:

phpversion(), para conocer la versión de PHP que utilizamos.

Componentes de una función

Para utilizar una función necesitamos conocer tres componentes:

  • Nombre de la función
  • Tipo de dato que genera
  • Lista de parámetros que recibe

Ejemplo

Supón una función llamada suma, que recibe dos enteros y genera (devuelve) otro entero:

  • Nombre: suma
  • Tipo de dato: numérico
  • Parámetros: $numero1 y $numero2, ambos de tipo integer

Funciones de PHP

PHP incorpora un gran número de funciones a nuestra disposición para ayudarnos con nuestras aplicaciones.

Por ejemplo, incluye funciones:

  • Matemáticas
  • Para manejo de ficheros
  • Para manejo de imágenes
  • De bases de datos
  • De arrays
  • De strings
  • ¡Y muchas más!

Documentación de funciones

Recordando: las funciones las describimos con 3 componentes: (1) nombre, (2) tipo de dato que devuelve y (3) parámetros.

Vamos a ver cómo leer la documentación de funciones de PHP:


Siguiendo con el ejemplo de la función suma, la describimos así:


	integer suma(integer $numero1, integer $numero2)

Solo con eso ya sabríamos cómo usar la función:


	$valor = suma(15, 201);

Pseudotipos documentando funciones

Hay funciones que permiten parámetros de varios tipos. Por ejemplo, la función suma podría aceptar también variables Float


$valor = suma(15.1, 202.09);

Para facilitar la documentación, se recurre a pseudotipos:

  • number - Indica que la variable puede ser Integer o Float
  • mixed - Indica que la variable puede ser de varios tipos (no necesariamente todos)

number suma(number $numero1, number $numero2)

Parámetros opcionales

Hay funciones cuyos parámetros son opcionales.

Por ejemplo, volviendo con la función suma(), supongamos que podemos pasarle opcionalmente 2 o 3 números:


$valor1 = suma(15, 202, 12);
$valor2 = suma(15, 202);

En este caso, el parámetro opcional iría entre corchetes:


number suma(number $numero1, number $numero2 [, number $numero3])

Parámetros ilimitados

Igualmente, hay funciones cuyo número total de parámetros es indeterminado.

Por ejemplo, la función suma() podría aceptar tantos valores como le proporcionemos:


$valor = suma(15, 202, 12, -5, -105, 1009, 5, 99);

Los parámetros obligatorios (los dos primeros al menos) irían como hasta ahora. El resto de parámetros opcionales los nombraríamos con '...' (tres puntos): 


number suma(number $numero1, number $numero2 [, number $...])

Ejemplo

Ya conocemos la función phpversion(). Vamos a ver cómo es en la documentación de PHP:

string phpversion ([ string $extension ] )


Vemos esta información:

  • Nombre: phpversion
  • Valor devuelto: string
  • Parámetros: un string opcional
