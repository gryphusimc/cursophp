¿Qué es una variable?

Se trata de una «casilla» donde guardamos un «dato».


Se componen de dos partes:

  1. Identificador: Un nombre con el que hacer referencia a esta variable
  2. Valor: Son los datos que va a almacenar la variable

Características

  • Tamaño: Fijo o variable.
  • Ámbito: Local o global.
  • Tipo: Numérico entero, número real, texto, etc.
  • Tipado: Estático o dinámico.


En PHP son de tamaño variable y de tipado dinámico. No solo no hay que definir el tipo, sino que una misma variable se podría modificar para que almacenara cualquier otro tipo.

Nombrar una variable en PHP

Se representan por un signo de dólar seguido por su nombre.


El nombre sigue algunas reglas:

  • Diferencia minúsculas y mayúsculas. $foo es distinto a $FOO.
  • Puede contener letras, números o guiones bajos
  • Pero debe comenzar por una letra o por un guión bajo.


Por ejemplo:


$altitud, $FechaNacimiento, $_direccion, $cotizacion2013 // Válidos
$2013cotizacion, fechaNacimiento, $valor1º // Inválidos

Crear una variable en PHP

Únicamente tenemos que asignarle un valor:


$nombre = "Hugo";

Utilizamos el operador '=', para asignar a la variable de la izquierda el valor que haya a la derecha.

Ahora, una vez que la variable está creada, ya podemos utilizarla:


echo $nombre; // Escribe el valor que tuviera la variable $nombre: Hugo

Tipos en PHP

  • Tenemos cuatro tipos escalares:
    • string: cadena de texto.
    • integer: número entero.
    • float: double, número real.
    • boolean: booleano, lógico, verdadero-falso.
  • Dos tipos compuestos:
    • array: matriz, arreglo.
    • object: objeto
  • Y otros tipos:
    • resource: recurso
    • NULL: valor nulo

String

Contienen una secuencia de cero o más caracteres. Varias formas. Vamos a ver dos.

Comillas simples

Consiste en encerrar la cadena de texto entre comillas simples:


$nombre = 'Juan';

¿Y si quisiéramos que el texto tuviera una comilla simple?


	$restaurante = 'Foster\'s Hollywood';

¿Y si queremos mostrar la barra?


	$dir = 'C:\\WINDOWS';

String

Comillas dobles

También podemos encerrar el texto entre comillas dobles.
Permite:

  • Expandir variables dentro del texto. 
    
$nombre = "Luis";
$saludo = "Hola $nombre!"; // Hola Luis!
  • Insertar caracteres escapados, como \n (cambio de línea) o \t (carácter tabulador). 
    
$nombre = "Luis";
$saludo = "Hola " . $nombre . "!\n"; // Hola Luis!\n

    Para concatenar dos textos usamos el operador '.' (punto):

Integer

Este tipo de variable permite almacenar un valor numérico sin decimales. Nunca hay que poner separador de miles.

Podemos usar operadores matemáticos:

  • + : Para la suma
  • - : Para la resta
  • * : Para la multiplicación
  • / : Para la división


Ejemplos:


$numero1 = 15;
$numero2 = -3;
$numero3 = 0;
$media = ($numero1 + $numero2 + $numero3) / 3; // La media es 4

Float

Este tipo de variable permite almacenar valores numéricos con decimales. Para separar decimales se utiliza el carácter '.'


Ejemplo:


$PI = 3.1415; // Valor de PI (Float)
$radio = 15; // Radio de un círculo (Integer)

// Podemos hacer operaciones matemáticas combinando Integer y Float
$area = $PI * $radio * $radio; // Área del círculo

Boolean

El tipo boolean es el más simple, y únicamente acepta dos valores: TRUE o FALSE


// Supongamos que hoy es martes
$ayer_fue_lunes = TRUE;
$ayer_fue_jueves = FALSE;

Resumen

Hemos aprendido:

  • Qué es y para qué sirve una variable.
  • Cómo son las variables en PHP y qué tipos hay.
  • Cómo utilizar los tipos escalares: string, integer, float y boolean.
