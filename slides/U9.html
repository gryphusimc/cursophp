Ejercicio 1

Crea un directorio llamado U9. Después, con tu editor de código, crea un fichero de tipo PHP llamado ejercicio1.php y guárdalo en la carpeta U9.

Una vez creado el fichero, añade el siguiente código HTML:


<DOCTYPE html>
<html>
<head>
	<title>Ejemplo</title>
</head>
<body>
</body>
</html>

Después, dentro de la etiqueta body, crea una variable de tipo string que contenga el valor "Hola", y otra de tipo float que contenga el valor "22.4", y ábrelo en tu navegador.

Ejercicio 2

Crea un nuevo fichero llamado ejercicio2.php y guárdalo en la carpeta U9. Añade de nuevo el código HTML.

Crea un array con estos 3 números: 15, 21, -7. Después calcula y muestra la media en el navegador.

Ejercicio 3

Crea un nuevo fichero llamado ejercicio3.php y guárdalo en la carpeta U9. Añade de nuevo el código HTML.

Averigua qué versión de PHP utilizas escribiendo el siguiente código PHP:


echo phpversion();

Crea una constante llamada VERSION y que tenga como valor un string con su número de versión. Si utilizas una versión de PHP igual o superior a la 5.3, utiliza el método nuevo.

Finalmente, convierte dicho valor en integer y muestra el resultado.

Ejercicio 4

Crea un nuevo fichero llamado ejercicio4.php y guárdalo en la carpeta U9. Añade de nuevo el código HTML.

Muestra en pantalla información acerca del navegador y la IP desde la que se está consultando.
