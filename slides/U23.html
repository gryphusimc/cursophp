Actividad 1

Los arrays son listas de valores. Estos valores pueden ser, a su vez, arrays. Supón el siguiente:


$usuarios = array(
	1 => array('nombre' => 'Juan', 'nacimiento' => '06/10/1988'),
	2 => array('nombre' => 'Laura', 'nacimiento' => '16/12/1993'),
	3 => array('nombre' => 'Daniel', 'nacimiento' => '22/09/1970'),
	4 => array('nombre' => 'Hugo', 'nacimiento' => '01/09/1982'),
	5 => array('nombre' => 'Elisa', 'nacimiento' => '27/02/1974'),
);

Crea una aplicación que muestre el nombre de los mayores de 25 años.

Actividad 2

Tenemos que programar una máquina dispensadora de refrescos. Calcula el cambio óptimo (usando el menor número de billetes/monedas posible).

Pasos:

  1. Crea dos variables. Una con el del precio del producto y otra con la moneda/billete con que pagan.
  2. Crea un array con todas las monedas/billetes que existen.
  3. Calcula el cambio.
  4. Calcula la mayor moneda/billete con la que puedes dar cambio.
  5. Resta al cambio el valor obtenido en el paso 4.
  6. Vuelve al paso 4, hasta que el cambio sea inferior que la menor de las monedas.

Actividad 3

En la actividad 2 no hemos tenido en cuenta un problema: ¿qué hacemos si nos falta alguna moneda? Actualiza el programa de manera que devuelva el cambio óptimo con las monedas disponibles. Si no fuera posible dar el cambio, mostrar un mensaje.

Actividad 4

Queremos hacer una aplicación multiplataforma que analice fechas que están en este formato de strftime:
«%d del %m del %Y». Averigua cómo hacerlo.

  • %d - día del mes con dos dígitos
  • %m - mes del año con dos dígitos
  • %Y - año con cuatro dígitos


¿Funcionaría tu solución con este formato de la función date?
«j \d\e\l n \d\e\l Y».

  • j - día del mes con uno o dos dígitos
  • n - mes del año con uno o dos dígitos
  • Y - Año con cuatro dígitos
