Curso de PHP

Funciones de arrays II

Por Jesús Ruiz-Ayúcar / @chuso

array_slice()

Extrae un subarray

array array_slice ( array $array , int $offset [, int $length = NULL [, bool $preserve_keys = false ]] )

Ejemplo:


$array1 = array(5,1,3,10,-2);
$array2 = array_slice($array1, 1, 2); // devuelve array(1,3)
$array3 = array_slice($array1, 0, 4); // devuelve array(5,1,3,10)

in_array()

Comprobar si contiene un valor

bool in_array ( mixed $needle , array $haystack [, bool $strict = FALSE ] )

Ejemplo:


$array1 = array(5,1,3,10,-2);
in_array(5, $array1); // true
in_array(0, $array1); // false

array_key_exists()

Comprobar si contiene una clave

bool array_key_exists ( mixed $key , array $array )

Ejemplo:


$array1 = array('a' => '1001', 'b' => '2031', 'd' => '4096');
array_key_exists('a', $array1); // true
array_key_exists('c', $array1); // false

Operaciones con arrays asociativos

array_keys()

Extraer todas las claves

array array_keys ( array $array [, mixed $search_value [, bool $strict = false ]] )

Ejemplo:


$array1 = array('a' => '1001', 'b' => '2031', 'd' => '4096');
$claves = array_keys($array1); // devuelve array('a','b','d');

array_values()

Extraer todos los valores

array array_values ( array $array )

Ejemplo:


$array1 = array('a' => '1001', 'b' => '2031', 'd' => '4096');
$valores = array_values($array1); // devuelve array('1001', '2031', '4096');

array_flip()

Intercambiar claves y valores

array array_flip ( array $trans )

Ejemplo:


$array1 = array('a' => '1001', 'b' => '2031', 'd' => '4096');
$array2 = array_flip($array1);
// devuelve array('1001' => 'a', '2031' => 'b', '4096' => 'd')

array_combine()

Unir dos arrays: claves-valores

array array_combine ( array $keys , array $values )

Ejemplo:


$claves = array('a', 'b', 'c');
$valores = array(1, 2, 3);
$claves_valores = array_combine($claves, $valores);
// Devuelve array('a' => 1, 'b' => 2, 'c' => 3);

Arrays y strings

implode()

Une los elementos de un array en un string

string implode ( string $glue , array $pieces )

Ejemplo:


$array1 = array(5,1,3,10,-2);
echo implode(", ", $array1);
// muestra "5, 1, 3, 10, -2"

$array2 = array(2,11,2013);
echo implode("/", $array2);
// muestra "2/11/2013"

explode()

Separa los elementos de un string en un array

array explode ( string $delimiter , string $string [, int $limit ] )

Ejemplo:


$string1 = "5, 1, 3, 10, -2";
$array1 = implode(", ", $array1);
// devuelve array(5,1,3,10,-2)

$string2 = "2/11/2013";
$array2 = explode("/", $string2);
// devuelve array(2,11,2013)

list()

Asigna valores de un array a variables. No es realmente una función

Ejemplo:


$array1 = array("a", "b", "c");

// Podemos asignar los valores a variables, de uno en uno:
$item1 = $array[0];
$item2 = $array[1];
$item3 = $array[2];

// O todos de golpe
list($item1, $item2, $item3) = $array1;

echo $item3." - ".$item2." - ".$item1;
// Muestra "c - b - a"

list($day, $month, $year) = explode("/", "31/12/1991");
echo "Hoy es $day del $month de $year";
// Muestra "Hoy es 31 del 12 de 1991"

Resumen:

  • Subarray: array_slice()
  • Comprobar si existe clave: array_key_exists()
  • Comprobar si existe valor: in_array()
  • Operaciones con arrays asociativos:
    • Extraer claves: array_keys()
    • Extraer valores: array_values()
    • Intercambiar claves-valores: array_flip()
    • Unir claves-valores: array_combine()
  • Arrays y strings:
    • De array a string: implode()
    • De string a array: explode()
  • Para asignar valores a variables: list()
