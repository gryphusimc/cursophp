Ficheros I

Por Jesús Ruiz-Ayúcar / @chuso

file_exists()

Comprueba si existe un fichero o directorio.

bool file_exists ( string $filename )

Por ejemplo, podemos comprobar si existe el directorio "/" (en Linux o Mac).


<?php
$ruta = "/"; // En Windows, pon "C:/";
if (file_exists($ruta)) {
	echo "Existe\n";
} else {
	echo "No existe\n";
}

is_dir()

Comprueba si la ruta proporcionada se corresponde con un directorio.

bool is_dir ( string $filename )

Ahora comprobamos si es un directorio.


<?php
$ruta = '/'; // En Windows, pon 'C:/'
if (is_dir($ruta)) {
	echo "Es un directorio\n";
} else {
	echo "No es un directorio\n";
}

is_file()

Comprueba si la ruta proporcionada se corresponde con un fichero (no un directorio).

bool is_file ( string $filename )

Ahora comprobamos si es un fichero.


<?php
$ruta = '/'; // En Windows, pon 'C:/'
if (is_file($ruta)) {
	echo "Es un fichero\n";
} else {
	echo "No es un fichero\n";
}

is_readable()

Comprueba si la ruta proporcionada se puede leer. Si es un fichero, podremos leer su contenido. Si es un directorio, podremos ver la lista de lo que contiene.

bool is_readable ( string $filename )

Vemos si podemos leer la ruta indicada.


<?php
$ruta = '/'; // En Windows, pon 'C:/'
if (is_readable($ruta)) {
	echo "Sí tenemos permisos de lectura\n";
} else {
	echo "No tenemos permisos de lectura\n";
}

is_writable()

Comprueba si la ruta proporcionada es escribible (tanto si es un fichero como si es un directorio).

bool is_writable ( string $filename )

Y lo probamos con el directorio raíz:


<?php
$ruta = '/'; // En Windows, pon 'C:/'
if (is_writable($ruta)) {
	echo "Sí tenemos permisos de escritura\n";
} else {
	echo "No tenemos permisos de escritura\n";
}

dirname()

Devuelve el nombre del directorio en el que se encuentra el fichero o directorio consultado.

string dirname ( string $path )

Vamos a probar esta función con el fichero que contiene el código (__FILE__ contiene la ruta del archivo en que se encuentra):


<?php
echo dirname(__FILE__)."\n";

filesize()

Devuelve el tamaño del fichero.

int filesize ( string $filename )

De nuevo, probamos esta función con el fichero que contiene el código:


<?php
echo filesize(__FILE__)."\n";

copy()

Copia un fichero a otra ruta.

bool copy ( string $source , string $dest [, resource $context ] )
Para mover un fichero, usar rename()

Para copiar un fichero, es aconsejable asegurarnos antes de que podemos:


<?php
$origen = __FILE__; // Fichero que vamos a copiar
$destino = dirname(__FILE__)."/copia.php"; // Copia del fichero

if (is_readable($origen) && is_writable(dirname($destino))) {
	copy($origen, $destino);
	echo "Copiado con éxito\n";
} else {
	echo "No se puede copiar\n";
}

rename()

Renombra (equivalente a mover) un fichero o directorio.

bool rename ( string $oldname , string $newname [, resource $context ] )

Para mover un fichero, es aconsejable asegurarnos antes de que podemos:


<?php
$origen = dirname(__FILE__).'/copia.php'; // Fichero que vamos a mover
$destino = dirname(__FILE__)."/nueva_copia.php"; // Nuevo nombre

if (is_writable($origen) && is_writable(dirname($destino))) {
	rename($origen, $destino);
	echo "Movido con éxito\n";
} else {
	echo "No se puede mover\n";
}

unlink()

Elimina un fichero

bool unlink ( string $filename [, resource $context ] )

De nuevo, comprobamos antes si podemos:


<?php
$archivo_para_eliminar = dirname(__FILE__).'/nueva_copia.php'; // Copia del fichero
if (unlink($archivo_para_eliminar)) {
	echo "Se ha eliminado con éxito\n";
} else {
	echo "No se ha podido eliminar\n";
}

mkdir()

Crea un directorio.

bool mkdir ( string $pathname [, int $mode = 0777 [, bool $recursive = false [, resource $context ]]] )

Vamos a crear una carpeta cualquiera:


<?php
$carpeta = dirname(__FILE__)."/prueba";

if (is_writable(dirname($carpeta))) {
	mkdir($carpeta);
	echo "Carpeta creada con éxito\n";
} else {
	echo "No se ha podido crear la carpeta\n";
}

rmdir()

Elimina un directorio (si estuviera vacío).

bool rmdir ( string $dirname [, resource $context ] )

Y ahora borramos la carpeta que creamos en el paso anterior:


<?php
$carpeta = dirname(__FILE__)."/prueba";

if (rmdir($carpeta)) {
	echo "Carpeta borrada con éxito\n";
} else {
	echo "No se ha podido borrar la carpeta\n";
}

disk_free_space()

Espacio libre del disco.

float disk_free_space ( string $directory )

disk_total_space

Espacio total del disco.

float disk_total_space ( string $directory ) 

<?php
$volumen = "/"; // En Windows, pon 'C:/';
echo disk_free_space($volumen)." libres de un total de ";
echo disk_total_space($volumen)."\n";

Resumen

  • ¿Existe? — file_exists()
  • ¿De qué tipo es? — is_dir() e is_file()
  • Permisos - is_readable() e is_writable()
  • Información — dirname() y filesize()
  • Operaciones habituales — copy(), rename() y unlink()
  • Operaciones con directorios — mkdir() y rmdir()
  • Información del disco — disk_free_space() y disk_total_space()
