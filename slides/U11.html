Condicionales

Por Jesús Ruiz-Ayúcar / @chuso

Problemática

Hasta ahora, el flujo de ejecución es secuencial: Se ejecuta una línea tras otra. A veces necesitaremos modificar esto, y lo lograremos mediante las estructuras de control.

Por ejemplo, a veces necesitaremos dividir el flujo de ejecución, de manera que se ejecute un código u otro en función de una condición. ¿Cómo lo haremos? Con condicionales:


si condicion
	entonces haz esto
En concreto: 

if ($condicion) {
	// haz esto
}

Condiciones

¿Qué es exactamente una condición?

Por ejemplo, para votar es necesario ser mayor de 18 años. Entonces la condición para votar es que la edad sea igual o mayor que 18.

¿Recuerdas los operadores de comparación?


$edad = 21;
define('EDAD_VOTAR', 18); // Constante, no varía

$es_mayor_de_edad = $edad >= EDAD_VOTAR; // TRUE

if ($es_mayor_de_edad) {
	echo "Puedes votar\n";
}

Añadir condiciones a la sentencia if

No es necesario que guardemos el valor de la condición en una variable. La mayoría de las veces ubicaremos la propia comprobación dentro de la sentencia if


$edad = 21;
define('EDAD_VOTAR', 18); // Constante, no varía

if ($edad >= EDAD_VOTAR) {
	echo "Puedes votar\n";
}

Si no se cumple la condición

Hay determinados casos en los que no desearemos también añadir código que se ejecute en caso de que la condición no se haya cumplido.


si condicion
	entonces haz esto
si no
	entonces haz esto otro
En concreto: 

if ($condicion) {
	// haz esto
} else {
	// haz esto otro
}

Ejemplo else

Continuando con el ejemplo de la mayoría de edad, tendríamos:


$edad = 21;
define('EDAD_VOTAR', 18); // Constante, no varía

if ($edad >= EDAD_VOTAR) {
	echo "Puedes votar\n";
} else {
	echo "Aún no puedes votar\n";
}

elseif

Quizás queramos realizar alguna comprobación previa antes de llegar a else.

El agua tiene 3 estados: sólido, líquido y gaseoso. Entre 0ºC y 100ºC está líquido. Por encima de 100ºC está gaseoso. Y por debajo de 0ºC está sólido.


$temperatura = 29;

if ($temperatura >= 100) {
	echo "Agua en estado gaseoso\n";
} elseif ($temperatura >= 0) {
	echo "Agua en estado líquido\n";
} else {
	echo "Agua en estado sólido\n";
}

Varios elseif

Veamos otro ejemplo. La atmósfera tiene 4 capas:

  1. Troposfera, por debajo de 15Km
  2. Estratosfera, entre 15 y 50Km
  3. Mesosfera, entre 50 y 80Km
  4. Termosfera, por encima de 80Km

$altitud = 51;

if ($altitud >= 80) {
	echo "Termosfera\n";
} elseif ($altitud >= 50) {
	echo "Mesosfera\n";
} elseif ($altitud >= 15) {
	echo "Estratosfera\n";
} else {
	echo "Troposfera\n";
}

Switch

Supongamos una serie de comparaciones else/elseif/else en la que solo utilizamos el operador de igualdad, por ejemplo:


$color = "red"; // Ponemos el nombre de un color en inglés

if ($color == "red") {
	echo "Rojo\n";
} elseif ($color == "green") {
	echo "Verde\n";
} elseif ($color == "white") {
	echo "Blanco\n";
} else {
	echo "No conozco ese color\n";
}

En este caso (todas las comparaciones son de igualdad) podremos utilizar una sintaxis alternativa utilizando switch (siguiente diapositiva)

Switch

El ejemplo anterior podríamos reescribirlo así:


$color = "red"; // Ponemos el nombre de un color en inglés

switch ($color) {
	case "red":
		echo "Rojo\n";
		// break sirve para salir del switch.
		// Si no lo pusiéramos, seguiría y escribiría "Verde"
		break;
	case "green":
		echo "Verde\n";
		break;
	case "white":
		echo "Blanco\n";
		break;
	default:
		echo "No conozco ese color\n";
}

Sintaxis alternativa

Algunas estructuras de control permiten una sintaxis alternativa, por ejemplo if, else y elseif:


if ($edad >= 18) {
	echo "Puedes votar\n";
} else {
	echo "Aún no puedes votar\n";
}

// Podría reescribirse como:
if ($edad >= 18):
	echo "Puedes votar\n";
else:
	echo "Aún no puedes votar\n";
endif;

Esta sintaxis alternativa se suele utilizar para generar plantillas de HTML

Ejemplo


<DOCTYPE html>
<html>
<head>
	<title>Ejemplo</title>
</head>
<body>
<?php $edad = 21; if ($edad >= 18): ?>
	<p>Para votar pulsa <a href='/votar.php'>aquí</a><p>
<?php else: ?>
	<p>Aún no puedes votar<p>
<?php endif; ?>
</body>
</html>
