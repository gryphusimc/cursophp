Conversión entre tipos

Podemos convertir el valor de una variable a otro tipo de dato.

Por ejemplo, podemos convertir un número (12) en texto ("12"): Son diferentes internamente.


$variable_convertida = CONVERSIÓN $variable_original;

Teniendo las siguientes opciones de conversión:

  • (int) o (integer) - Convertir a integer
  • (bool) o (boolean) - Convertir a boolean
  • (float), (double) o (real) - Convertir a float
  • (string) - Convertir a string
  • (array) - Convertir a array (está sin definir aún)
  • (object) - Convertir a object
  • (unset) - Convertir a NULL

Conversión entre tipos

Ejemplos


$edad_string = "25"; // Es de tipo string (está entre comillas)
$edad_int = (integer) $edad_string; // $edad_int vale 25 (integer).

$pi = 3.14; // De tipo float
$pi_redondeado = (integer) $pi; // Vale 3, de float a integer

$es_siglo_21 = TRUE; // Vale TRUE, luego es boolean
$es_siglo_21_string = (string) $es_siglo_21; // ¡Se convierte a "1"!

$pelicula = "12 monos"; // De tipo string
$pelicula_integer = (int) $pelicula; // 12

$frase = "Tengo 105"; // De tipo string
$frase_integer = (int) $frase; // 0

Variables predefinidas

PHP incorpora un gran número de variables predefinidas (creadas automáticamente). Éstas son las principales (de tipo array, contienen muchos valores):

  • $_SERVER - Variables del entorno de ejecución
  • $_GET - Variables recibidas por HTTP GET
  • $_POST - Variables recibidas por HTTP POST
  • $_FILES - Variables con ficheros cargados por HTTP
  • $_REQUEST - Combina $_GET, $_POST y $_FILES
  • $_COOKIE - Variables de las cookies HTTP
  • $_SESSION - Variables de sesión PHP
  • $_ENV - Variables de entorno

Variables predefinidas

Ejemplos


// Muestra información sobre el navegador que realizó la petición
echo $_SERVER['HTTP_USER_AGENT'];

// Muestra la IP del cliente
echo $_SERVER['REMOTE_ADDR'];


Más adelante veremos formularios, cookies y sesiones, y utilizaremos $_POST, $_GET, $_FILES, $_COOKIE y $_SESSION.

Constantes

Similar a una variable, pero su valor no se puede modificar. A la hora de nombrarlas se siguen las mismas reglas que con las variables, solo que no comienzan por $. Por convenio se suelen utilizar solo letras mayúsculas.

Sintaxis:


define("NOMBRE", valor);

// Ejemplo:
define("PI", 3.1415);

Sintaxis nueva (desde PHP 5.3):


const NOMBRE = valor;

// Ejemplo
const PI = 3.1415;

Constantes predefinidas

Algunas son:

  • __LINE__ : Línea actual del fichero
  • __FILE__ : Fichero actual
  • __DIR__ : Directorio actual

// Suponemos que este fichero está en /var/www/U8/ejemplo.php
echo __FILE__; // Muestra /var/www/U8/ejemplo.php
echo __DIR__; // Muestra /var/www/U8
echo __LINE__; // Muestra 4
