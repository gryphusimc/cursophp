Leyendo y escribiendo ficheros

Por Jesús Ruiz-Ayúcar / @chuso

Leeyendo el contenido de ficheros

Vamos a ver diferentes métodos, ninguno basado en recursos:

  • file_get_contents() - Lee y retorna el contenido
  • file() - Lee y retorna el contenido en un array
  • readfile() - Lee y muestra el contenido

file_get_contents()

Lee y devuelve el contenido de un fichero en un string.

string file_get_contents ( string $filename [, bool $use_include_path = false [, resource $context [, int $offset = -1 [, int $maxlen ]]]] ) 

<?php
$fichero = "prueba.txt";

if (file_exists($fichero) && is_readable($fichero)) {
	// Guardamos en $texto el contenido del fichero
	$texto = file_get_contents($fichero);
	echo nl2br($texto);
}

file()

Lee y devuelve el contenido de un fichero en un array de strings, un elemento por línea.

array file ( string $filename [, int $flags = 0 [, resource $context ]] ) 

<?php
$fichero = "prueba.txt";

if (file_exists($fichero) && is_readable($fichero)) {
	// Guardamos el contenido del fichero en el array $texto_array
	$texto_array = file($fichero);

	foreach($texto_array as $linea) {
		echo $linea."
";
	}
}

readfile()

Lee y envía a la salida estandar (como echo) el contenido de un fichero.

int readfile ( string $filename [, bool $use_include_path = false [, resource $context ]] ) 

<?php
$fichero = "prueba.txt";

if (file_exists($fichero) && is_readable($fichero)) {
	header("Content-Type: text/plain");

	// Muestra directamente el contenido de $fichero
	readfile($fichero);
}

Nota: Aún no hemos visto la función header(). En este caso indicamos que el contenido no es HTML, sino texto plano.

Escribiendo el contenido de un fichero

file_put_contents()

int file_put_contents ( string $filename , mixed $data [, int $flags = 0 [, resource $context ]] )

Consulta los flags disponibles.


<?php
$fichero = "prueba.txt";
date_default_timezone_set("Europe/Madrid");

if (file_exists($fichero) && is_writeable($fichero)) {
	$linea = "Son las ".date("H:i:s")."\n";
	file_put_contents($fichero, $linea, FILE_APPEND);

	header("Content-Type: text/plain");

	// Muestra directamente el contenido de $fichero
	readfile($fichero);
}

Wrappers

Cuando abrimos un fichero, como por ejemplo éste,

/var/www/dominio.com/pictures/pic039.jpg

es como si hiciéramos esto:

file:///var/www/dominio.com/pictures/pic039.jpg


¿Qué es ese file:// que precede al nombre del fichero?

Es un wrapper (envoltura).

Wrappers predeterminados

PHP incluye un gran número de wrappers:

  • file:// - Sistema local de archivos.
  • http:// - Acceso a URL mediante HTTP(S).
  • ftp:// - Conexión a un servidor FTP(S)
  • php:// - Flujos de E/S.
  • ssh2:// - Conexión mediante SSH2.
  • zip:// - Ficheros comprimidos con ZIP.


Podemos usarlos con todas las funciones de ficheros.

stream_get_wrappers()

Devuelve un array con los wrappers disponibles en nuestro sistema.

array stream_get_wrappers ( void ) 

<?php

print_r(stream_get_wrappers());

Ejemplos con wrappers


<?php
// leemos del wrapper http://
$contenido = file_get_contents("http://www.google.com");

// Escribimos sin indicar wrapper (por omisión, file://)
file_put_contents("google.html", $contenido);

// Leemos a partir del wrapper file://
$contenido1 = file_get_contents("file:///Users/chuso/hubiC/Ejemplos/google.html");

// Escribimos en ftp://
@file_put_contents("ftp://usuario:pass@host.com/google.html", $contenido1);

echo htmlentities(utf8_encode($contenido1));
Fíjate que hemos puesto una @ antes de file_put_contents(). Sirve para que no se muestren errores en caso de que los hubiera.
