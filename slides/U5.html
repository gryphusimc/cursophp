Ficheros PHP

Deben tener la extensión .php (en minúsculas)


Nombre predeterminado: index.php


Si en el directorio raíz existe index.php, estas dos URL son equivalentes:

  • http://example.com/
  • http://example.com/index.php

Etiquetas PHP

El código PHP va encerrado entre estas etiquetas: <?php y ?>

Fuera de esas etiquetas puede haber código HTML.

Como en el ejemplo que vimos en la Unidad 4:


<!DOCTYPE html>
<html>
<head>
	<title>Ejemplo</title>
</head>
<body>
	<?php echo "¡Hola, Mundo!"; ?>
</body>
</html>

Hay más etiquetas PHP, como veremos ahora.

Mostrar algo en pantalla

Normalmente usaremos echo. Por ejemplo:


<?php echo "¡Hola!"; ?>

O bien podemos usar estas otras etiquetas de PHP:


<?="¡Hola!"?>

También podemos mostrar código HTML: 


<?php SI (CONDICION): ?>

Se cumple la condición

<?php SI NO: ?>

No se cumple la condición

<?php FIN SI ?>

Profundizaremos en las condiciones en posteriores unidades.

Separación de instrucciones

Una instrucción es la unidad mínima de funcionamiento en un lenguaje de programación

Puede ser, por ejemplo, realizar una operación matemática o utilizar el comando 'echo'.

Siempre irán separadas por el caracter punto y coma ";".

Para no olvidarnos, siempre pondremos un ";" al final de cada instrucción.


echo "1";
echo "2";
echo "3";

Comentarios

Son anotaciones que escribimos en el código, pero que no realizan ninguna acción. Recuerda que otras personas podrían en algún momento tener que revisar lo que tú has programado.


Muy útil para:

  • Añadir documentación técnica sobre nuestro código.
  • Explicar para qué sirve un determinado bloque.
  • O explicar cómo funciona.
  • Dejar por escrito algo pendiente de realizar.

Tipos de comentarios

De una línea

Comienzan con // y acaban al finalizar la línea:


echo "hola";
// Esto es un comentario
echo "adiós"; // Esto es otro comentario
// echo "¿y yo soy un comentario?";

De varias líneas

Comienzan con /* y terminan en la primera aparición de */.


echo "hola";
/* Esto es un comentario */
echo "adiós"; /* Esto es otro comentario
// echo "¿y yo soy un comentario?" */ echo "¿Y yo?";
