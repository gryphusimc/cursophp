Averiguar si es un número

Dada la naturaleza dinámica de PHP, no podemos garantizar que una variable va a tener el tipo de dato que deseamos.


$num1 = "#12"; // comienza con '#', luego se convierte a 0
$num2 = 10;
echo suma($num1, $num2); // ¡A suma() le llega un string!

Tenemos 3 funciones:

  1. is_numeric()
  2. is_int(), is_integer() o is_long()
  3. is_float(), is_double() o is_real()

is_numeric()

Comprueba si una variable es un número o un string numérico.

Resumen:

bool is_numeric ( mixed $var )

is_numeric(12); // TRUE
is_numeric(1.13); // TRUE
is_numeric(112,3); // FALSE y Warning
is_numeric("61"); // TRUE
is_numeric("#61"); // FALSE
is_numeric("doce"); // FALSE

is_int()

Comprueba si una variable es un número entero.

bool is_int ( mixed $var )

is_int(12); // TRUE
is_int(1.13); // FALSE
is_int(112,3); // FALSE y Warning
is_int("61"); // FALSE

is_integer() e is_long() son equivalentes.

is_float()

Comprueba si una variable es un número real

bool is_float ( mixed $var )

is_float(12); // FALSE
is_float(1.13); // TRUE
is_float(112,3); // FALSE y Warning
is_float("61.3"); // FALSE

is_double() e is_real() son equivalentes.

decbin()

Función que convierte de decimal a binario.

string decbin ( int $number )

echo decbin(12); // Muestra 1100
echo decbin(20000); // Muestra 100111000100000

bindec()

Función que convierte de binario a decimal.

number bindec ( string $binary_string )

echo bindec("1100"); // Muestra 12
echo bindec("100111000100000"); // Muestra 20000

decoct()

Convierte un valor entero a un string con su representación octal.

string decoct ( int $number )

$numero1 = 12; // Asigna 12
$numero2 = 014; // Asigna 12
echo decoct($numero1); // Muestra "14"
echo decoct($numero2); // Muestra "14"

octdec()

Convierte un string con un valor octal a un entero decimal.

number octdec ( string $octal_string )

octdec("8"); // Devuelve un 8 entero
octdec("14"); // Devuelve un 12 entero

dechex()

Convierte un valor entero a un string en hexadecimal

string dechex ( int $number )

echo dechex(15); // Muestra "A";
echo dechex(9);  // Muestra "9";

hexdec()

Convierte una representación hexadecimal a su valor decimal

number hexdec ( string $hex_string )

echo hexdec("FFFFFF"); // Muestra 16777215
echo hexdec("10"); // Muestra "16"

Precisión de los números

Una variable de tipo integer de PHP no puede contener un valor arbitrariamente grande. Su valor viene limitado por la plataforma sobre la que funciona.

Precisión según plataforma

Plataformas de 32 bits Plataformas de 64 bits
2147483647 9223372036854775807

Si se sobrepasan esos valores, se convierten a Float.

En dichos casos es recomendable recurrir a las funciones BC Math o GMP.

base_convert()

Convierte un número entre dos bases

string base_convert ( string $number , int $frombase , int $tobase )

echo base_convert("11", 2, 10); // Muestra un 3
echo base_convert("10", 10, 16); // Muestra un A
echo base_convert("22", 7, 6); // Muestra un 24

floor()

Redondea a la baja un número real

float floor ( float $value )

echo floor(2); // Muestra 2
echo floor(2.5); // Muestra 2
echo floor(2.3); // Muestra 2
echo floor(2.9); // Muestra 2

Es equivalente a (int) $value.

ceil()

Redondea al alza un número real

float ceil ( float $value )

echo ceil(2); // Muestra 2
echo ceil(2.5); // Muestra 3
echo ceil(2.3); // Muestra 3
echo ceil(2.9); // Muestra 3

round()

Redondea al entero más próximo

float ceil ( float $value )

echo round(2); // Muestra 2
echo round(2.3); // Muestra 2
echo round(2.5); // Muestra 3
echo round(2.9); // Muestra 3

rand()

Genera un número pseudoaleatorio

int rand ( int $min , int $max )

echo rand(0, 10); // Genera un número aleatorio entre 0 y 10
echo rand(0, 60); // Genera un número al azar entre 0 y 60

Restablece el generador de números aleatorios

void srand ([ int $seed ] )

Devuelve el rango máximo de números aleatorios

int getrandmax ( void )

pi()

Devuelve el número Pi

float pi ( void )

En su lugar se podría utilizar la constante predefinida M_PI

pow()

Elevar un número a la potencia indicada

number pow ( number $base , number $exp )

Para elevar 2 a la 8 (2^8) podemos: 


// Opción 1
echo 2*2*2*2*2*2*2*2; // Poco práctico

// Opción 2
$num = 1;
for ($i=0; $i< 8; $i++) {
	$num *= 2;
}
echo $num; // Muy engorroso

// Opción 3
echo pow(2, 8);
Desde PHP 5.6 puedes poner 2**8 para 28.

float sqrt ( float $arg )

Funciones trigonométricas

  • cos() y cosh() - Coseno y coseno hiperbólico
  • sin() y sinh() - Seno y seno hiperbólico
  • tan() y tanh() - Tangente y tangente hiperbólica
  • acos() y acosh() - Arcocoseno y arcocoseno hiperbólico
  • asin() y asinh() - Arcoseno y arcoseno hiperbólico
  • atan() y atanh() - Arcotangente y arcotangente hiperbólica
  • deg2rad() - Grados a radianes
  • rad2deg() - Radianes a grados

Otras funciones

  • abs() - Valor absoluto de un número
  • min() - Valor mínimo de una lista o de un array
  • max() - Valor máximo de una lista o de un array
  • log() - Logaritmo neperiano
